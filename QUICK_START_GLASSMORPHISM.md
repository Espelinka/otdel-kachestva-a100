## 📋 QUICK TROUBLESHOOTING

### Glass effect not showing?
- ✅ Use modern browser (Chrome 76+)
- ✅ Enable hardware acceleration
- ✅ Update graphics drivers

### Background not visible?
- ✅ Check Fon.jpg exists
- ✅ Check path in CSS (../Fon.jpg)
- ✅ Use colorful image

### Animations laggy?
- ✅ Close other tabs
- ✅ Update browser
- ✅ Check GPU settings

### Text hard to read?
- ✅ Increase screen brightness
- ✅ Use darker background image
- ✅ This is optimized for visibility
